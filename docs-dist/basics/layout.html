    # Row Props

    参数 说明 类型 可选值 默认值
    gutter 间隔 Number — 0
    align 纵轴的对齐方式 String — center
    flex 是否开启flex布局 Boolean true false
    justify 横轴的对齐方式 String — space-between

    # Col Props

    参数 说明 类型 可选值 默认值
    span 列元素宽度 Number — 24
