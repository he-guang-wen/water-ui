    # Collapse Props

    参数 说明 类型 可选值 默认值
    v-model 需要展开的列表 Boolean true false
    arrow 是否显示箭头 Boolean false true
    accordion 是否开始手风琴效果 Boolean true false
    background 背景颜色 String — #fff

    # CollapseItem Props

    参数 说明 类型 可选值 默认值
    name 标识符 String — —
    title 左侧侧标题 String — —
    value 右侧内容 String — —

    # CollapseItem Events

    事件名 说明 回调参数
    click 点击 event: Event
    open 打开 —
    close 关闭 —
