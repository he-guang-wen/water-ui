# Divider 分割线

### 基础用法
```js
<template>
    <div>
        <wr-divider>已经见底了</wr-divider>
    <div>
</template>
```

### 文字颜色
```js
<template>
    <div>
        <wr-divider color="red">已经见底了</wr-divider>
    <div>
</template>
```

### 线条颜色
```js
<template>
    <div>
        <wr-divider border-color="red">已经见底了</wr-divider>
    <div>
</template>
```

##  Props

<md-table-warp>
| 参数         | 说明                   | 类型    | 可选值                | 默认值 |
|:-------------|:-----------------------|:--------|:----------------------|:-------|
| color     | 文字颜色             | String | —                  | #000  |
| border-color         | 线条颜色 | String | true                  | #ccc  |
                                
</md-table-warp>
